# **Platform as a Service**
# **Beschreibung**

Man stellt eine Plattform bereit, auf der die Kunden eigene Anwendungen entwickeln, bereitstellen und verwalten können, ohne sich um die zugrunde liegene Infrastruktur kümmern zu müssen

# **Beispiele**
- Oracle Cloud Platform (Datenbankmanagement)
- Mendix: (Lowcode-Plattform)
- Google App Engine (Entwicklungs-Plattform)
- Red Hat OpenShift (Kubernetes-Plattform)
- AWS Lambda (Serverlose Funktion als Service)
